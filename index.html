Etiquetas extras

Etiqueta pre

La etiqueta pre sirve para respetar espacios y lineas en blanco, al contrario de la etiqueta p

hola soy maru tengo 17 casi 18 

 Hola    soy         Maru      tengo 17    casi       18

Etiqueta abr

La etiqueta abr se utliliza para las abreviaturas

Segun datos de la abr OMS el consumo excesivo de azucar provoca daños severos en la salud

Etiqueta bdo

Si se desea cambiar la direccion de un texto, se utiliza la etiquta bdo

Este es un texto normal

Este es un texto bizarro

Etiqueta cite

La etiqute cite se utliliza para escribir citas textuales

Me voy a casar con Eduardo Luis Nader por: Maru

Etiqueta Small

La etiqueta small hace el texto mas pequeño

Este texto esta adentro de una etiqueta small

Etiqueta sub

Si se desea escribir un subíndice, se debe utlizar la etiqueta sub

143 me gusta eduaro luis

Etiqueta sup

Si se desea escribir un superíndice, se debe utlilizar la etiqueta sup

144 lololo

Etiqueta time

Para describir fechas y horas se debe utlizar la etiqueta time

Mañana es dia 2024-11-06

Mañana es dia 6 de noviembre de 2024

Etiqueta adress

Para especificar al navegador que el texto se refiere a una dirección utliliza la etiqueta adress

Loma doradda No. 100 Colonia Lomas de Rosales CP 89100 Tampico, Tams.
